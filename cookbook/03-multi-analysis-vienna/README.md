# 03 — All 8 analyses in one run

Runs every supported analysis (wind, PWC, daylight, sun hours, SVF, solar radiation, UTCI, TCS) over the same Vienna polygon and renders a multi-panel Plotly HTML for side-by-side comparison.

**Run:** `python multi_analysis.py`
**Deps:** `infrared-sdk plotly numpy python-dotenv`
**Note:** longer runtime — multiple analyses dispatched.
